## BLOQUE 2 — THE FIVE FORMS ON ONE PAGE (40') — DE PIE ~90%

**Objective: everyone writes/produces all five forms in order, correct, no present leak.**

1. Board — la cadena de las 5 formas:
```
THE WHOLE PAST ON ONE PAGE — 5 forms, no present leak
(1) OLD HABIT/STATE    used to + base  /  would + base   (action only)
(2) BACKGROUND         was/were + -ing
(3) THE EVENT          when + past simple  /  past simple
(4) THE EARLIER EVENT  had + participle
(5) THE ORDER          first / then / after that / while / finally
Self-check: bare after used to? -ing background? bare-past event?
  participle after HAD? order words present? ANY present verb? -> fix.
```
2. Señala la logica: *"Nothing new today. Every form is one you already own — now they run in one clean line, no leak to the present."*
3. **Mastery relay (14', DE PIE):** historia colaborativa en el board de 20+ clausulas; cada aporte se ETIQUETA con su forma (1-5) antes de escribirse. Lectura coral al final + conteo de formas usadas.
4. **Errores tipicos al board (coro 2x cada correccion):**

   - "I had went" -> "I had GONE" (participio)

5. **Zone walk (8', DE PIE):** las 5 formas como zonas. Dices una linea suelta; las parejas CAMINAN a la forma correcta Y la reproducen limpia en voz alta.
