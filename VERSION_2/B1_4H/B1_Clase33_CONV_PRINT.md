## BLOQUE 2 — THE RIGHT LEVEL (40')

**Objective: everyone judges one real work situation using so / too / enough — fairly, without exaggeration.**

1. Meta en 1 linea (arriba). **Avance de proyecto hoy: una linea para MY STORY — "I'm strong enough now: I was too ___ before, but not anymore."**
2. Board — el molde oral (Grammar lo formaliza hoy mismo con M30; aqui NO se enseña la regla):
```
JUDGING THE LEVEL FAIRLY
"The plan is good enough to start — we don't need more."
"This deadline is too tight. It's not realistic."
"She is so organized! Everything is ready."
"The report isn't clear enough yet — one more pass."
"I love this client, but they're too demanding sometimes."
(descriptor + ENOUGH / not ... ENOUGH ; SO = high & good ;
 TOO = excessive / not wanted)
```
3. Modela 45 seg juzgando una situacion real tuya: algo que esta "good enough", algo "too much", algo "so good".
4. Preparacion en parejas (7'): cada uno elige 2 situaciones reales de su trabajo — una que esta al nivel justo (enough) y una excesiva (too).
5. Ejecucion (15'): mini-evaluaciones en parejas: A juzga ("This process is efficient enough"); B matiza con justicia ("Maybe, but the last step is too slow"). Se busca el nivel JUSTO, no la queja. Cambio de pareja a mitad.
6. Producto verificable (10'): vuelta completa — cada estudiante juzga al grupo UNA situacion real con so/too/enough (20 seg). Tu cuentas moldes limpios en el board.
7. Feedback: logro primero (¿el juicio es justo?), forma despues. Errores 2+ parejas -> board anonimo. Nombres a tu cuaderno.
