## BLOQUE 2 — SO / TOO / ENOUGH (40') — DE PIE ~90%

**Objective: everyone builds so/too/enough sentences on paper, judging real work levels.**

1. Board — cita del libro (M30, Guia 1 p.268):
```
REQUIRED LEVELS (A2 Book M30, p.267-269)
  "enough" va DESPUES del descriptor:
     "I am tall enough."   "I am not tall enough."
  "so" = nivel alto, pero bueno:  "You are so tall!"
  "too" = excesivo / no deseable: "You are too tall."
BOOK EXAMPLE PAIRS (Ejemplos 1, p.269):
  "She is so nice!"  /  "She is too nice." (can't fight)
  "It is so new!"    /  "It is too new." (I like classic cars)
  "He is so smart!"  /  "He is too smart." (can't teach kids)
  "You are so early." /  "It is too early." (class at 5:00)
  "The party was so late!" / "I was too late." (couldn't travel)
```
2. **Meta de tarea comunicativa (1 linea):** *"By the end of this block, this board holds 12 real work judgments — 4 with 'enough', 4 with 'so', 4 with 'too'."*
3. **Preparacion en parejas (7', DE PIE junto al board):** cada pareja arma 1 frase con "enough" (incl. una "not ... enough"), 1 con "so", 1 con "too" — reales de su trabajo.
4. **Ejecucion (12'):** cada pareja ESCRIBE sus frases en el board, las dice, y el grupo verifica: ¿"enough" DESPUES del adjetivo? ¿"so" para lo bueno-alto y "too" para el exceso? Meta: 12+ limpias.
5. **Errores tipicos (board, coro 2x):**
   - "enough good" -> "good enough" (enough va despues)
   - "so much organized" -> "so organized" (so + adjetivo, sin much)
   - "too demanding" bien; "too much demanding" -> "too demanding"
   - "very enough" -> "enough" (no se intensifica)
6. **Producto verificable:** lectura coral del board; feedback primero sobre el logro, luego la forma.
