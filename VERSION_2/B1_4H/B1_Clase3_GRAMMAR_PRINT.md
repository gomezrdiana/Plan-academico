## ANEXO — M4 + M5 verbatim + Plan B (no leer en clase)

**A1 Book, Module 4 — "Where Should We Live? / Giving Recommendations" (p.41-44), Guia 1 — "Should" (verbatim):**
1. La auxiliar "should" (pronunciada "shud") indica "deberia", y es usada para hacer recomendaciones.
2. Esta palabra se trata exactamente como "can". -> "Juan should play soccer." = "Juan deberia jugar futbol."
3. En su forma abreviada "should + not" = "shouldn't" (pronunciada "shudunt"). -> "Juan should not play soccer."
4. La palabra de pregunta "where" significa "donde". -> "Where should Juan play?"
5. Igual a "can", la respuesta corta es simplemente "should" y "shouldn't".
**Ejemplos (p.44):** "Where should Martin live?" -> "He should live in London." · "Should Freddy live in Colombia?" -> "Yes, he should." / "No, he shouldn't." · "Should the dog live in Bucaramanga?" -> "Yes, it should." / "No, it shouldn't."

**A1 Book, Module 5 — "Numbers to 100 / Count 'em!" (p.61-64), Guia 1 (verbatim):**
- LA HORA: dos grupos, sin "and". "2:10" -> "two ten"; "4:45" -> "four forty-five". Minutos 01-09 incluyen "oh": "2:08" -> "two oh eight". "o'clock" para hora en punto: "6:00" -> "six o'clock".
- LOS PRECIOS: dos grupos, dolares y centavos. "$7.35" -> "seven dollars and thirty-five cents"; a menudo se omite y se dice "seven thirty-five"; "$13.05" -> "thirteen oh five".
- Vocabulario 0-100: 13 thirteen ... 30 thirty, 40 forty, 50 fifty ... 100 one hundred.
