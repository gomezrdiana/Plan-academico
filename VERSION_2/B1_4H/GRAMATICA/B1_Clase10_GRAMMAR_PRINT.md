## BLOQUE 2 — LA TABLA DE PRONOMBRES OBJETO (40') — DE PIE ~90%

**Objective: everyone replaces a noun with the correct object pronoun, keeping the order, fast.**

1. Board — cita del libro (M16, p.197-201):
```
OBJECT PRONOUNS (A1 Book M16, p.197-201)  — after the verb / after a preposition
   SUBJ -> OBJ:  I->ME  you->YOU  he->HIM  she->HER  it->IT  we->US  they->THEM
   "I love Ana." -> "Ana loves ME."
   "Do you talk with Omar?" -> "Yes, I talk with HIM."
   "Can you help my mother?" -> "Yes, I can help HER."
   "Do you like pizza?" -> "Yes, I like IT."
   "Do you like hamburgers?" -> "Yes, I like THEM."
Rule (book): al reemplazar un sustantivo con un pronombre objeto, se mantiene
   el ORDEN original de la frase. THEM = personas, animales y objetos.
   *** NEVER "he helped I" — the object slot takes ME. ***
```
2. Señala la logica: *"After the verb, after a preposition — never in the subject slot. 'He helped ME', not 'he helped I'. 'Her called me' is wrong — that slot is SHE. Position decides the form."*
3. **Drill en circulo (10', DE PIE):** tu lanzas una frase con nombre ("I know Camilo") -> estudiante 1 la reemplaza ("I know HIM") -> estudiante 2 la niega ("I don't know him") -> estudiante 3 la pregunta ("Do you know him?"); el 4 lanza la siguiente. Velocidad creciente.
4. **Errores tipicos al board (coro 2x):**
   - "he helped I" -> "he helped ME"
   - "her called me" -> "SHE called me" (sujeto)
   - "I like they" -> "I like THEM"
   - "I talk with he" -> "I talk with HIM" (tras preposicion)
5. **Zone walk (12', DE PIE):** tablero `SUBJECT (before verb) | OBJECT (after verb/prep)`. Tu dices un pronombre en una frase; las parejas CAMINAN a la zona correcta segun su posicion Y producen la frase. El que llega justifica.
6. Cierre del bloque: produccion libre de pie — cada uno 2 frases verdaderas reemplazando nombres reales por object pronouns.
