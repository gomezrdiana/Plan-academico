## BLOQUE 2 — USED TO / WOULD EN PAPEL (40') — DE PIE ~90%

**Objective: everyone forms affirmative, negative, question with USED TO + bare, and uses WOULD only for actions.**

1. Board:
```
USED TO / WOULD — the past you left behind
(+)  subj + USED TO + BARE verb     "I used to work alone."  (habit OR state)
(-)  subj + DIDN'T USE to + BARE     "I didn't use to speak up."  (no -d!)
(?)  DID + subj + USE to + BARE       "Did you use to smoke?"  (no -d!)
     Short: "Yes, I did. / No, I didn't."
WOULD + BARE = repeated ACTION only (NOT a state):
     "Every winter we would travel."  (action = OK)
     "I would be nervous" -> WRONG -> "I used to be nervous."  (state)
Master rule: after USED TO / DID / DIDN'T -> the verb is BARE.
```
2. Señala la logica: *"Same bare rule as after DID (Cl 13). USED TO always = bare. The -d dies in the negative and the question."*
3. **Drill en circulo (10', DE PIE):** cadena — tu lanzas sujeto + verbo ("she / smoke") -> est. 1 afirmativa used to, 2 negativa, 3 pregunta, 4 short answer; el 5 lanza el siguiente. Velocidad creciente.
4. **Errores tipicos al board (coro 2x cada correccion):**
   - "I used to worked" -> "I used to WORK" (bare)
   - "I didn't used to" -> "I DIDN'T USE to" (no -d)
   - "I would be shy" -> "I USED TO be shy" (would no va con estados)
5. **Zone walk (12', DE PIE):** tablero `ACTION (used to / would) | STATE (used to ONLY)`. Tu dices un verbo ("live abroad" / "arrive early" / "hate coffee"); las parejas CAMINAN a la zona correcta Y producen la frase con la forma valida en voz alta. El que llega, justifica en 1 linea.
6. Cierre del bloque: produccion libre de pie — cada uno 2 lineas verdaderas "I used to..." sobre su vida laboral/estudio. El uso libre cierra el bloque.
