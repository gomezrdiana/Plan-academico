## BLOQUE 2 — QUESTION TAGS (40') — DE PIE ~90%

1. Board — cita del libro (M1, Guia 1 p.3):
```
QUESTION TAGS (B1 Book M1, p.3-4)
Rule: "The question tag uses the SAME auxiliary as the
      principal statement." Affirmative -> negative tag;
      negative -> affirmative tag. Almost always contracted.
  to be:    "You are sick, aren't you?" / "You aren't sick, are you?"
  I am:     "I am very tall, AREN'T I?"  (amn't does not exist)
```

## ANEXO — VERBATIM DEL LIBRO + PLAN B + NOTA INTERNA

### M1 — Question Tags (Guia 1 p.3; Ejemplos p.4; Ejercicios p.5)
Regla verbatim p.3: *"The question tag uses the same auxiliary as the principal statement. If the statement is affirmative, the tag is negative... If the statement is negative, the tag is affirmative. Note: The question tag almost always uses the contraction."* — *"I am very tall, aren't I? (Amn't does not exist, so we use aren't for I)."* — do/does/did: *"Sam plays soccer, doesn't he?"*
Ejemplos p.4: *"You can see me, can't you?"* · *"Fabricio came to the school, didn't he?"* · *"You would tell me, wouldn't you?"* · *"Ana Maria wants to go, doesn't she?"*

### M2 — Either / Neither / Both (Guia 1 p.9; Guia 2 p.13; Guia 3 p.17)
p.9: *"We use 'either' to say 'o' or 'cualquier'. It indicates TWO options."* — *"Either of the boys can help you."*
p.13 IMPORTANT: *"'Neither…nor' are never used with a negative 'not'."* — *"Neither of the men should arrive today."* — tambien: *"John doesn't have a car. Neither do I."*
p.17: *"'Both…and' is the equivalent to 'ambos…y'... we do not use 'both' to say 'tambien'; we use 'also' or 'too'."* — *"Both of the boys can help you."*
