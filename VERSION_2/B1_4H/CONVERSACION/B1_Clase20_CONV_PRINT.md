## BLOQUE 2 — THE FIVE FORMS IN ONE STORY (40')

**Objective: by the end, everyone tells a 10-line true story using all five forms, no present leak.**

1. Anuncia la meta en 1 linea (arriba). Board — la cadena de las 5 formas (uso oral):
```
THE WHOLE PAST — 5 forms, one story, no present leak
(1) OLD HABIT/STATE   used to + base / would + base
(2) BACKGROUND        was/were + -ing
(3) THE EVENT         when + past simple / past simple
(4) THE EARLIER EVENT had + participle
(5) THE ORDER         first / then / after that / while / finally
Self-check card (5 questions): habit? background -ing? event bare past?
  earlier event = had + participle? order words present? ANY present verb? -> fix it.
```
2. Modela 60 seg: tu historia mas dura de principio a fin, las 5 formas. Los estudiantes marcan con los dedos cada forma al oirla.
3. Preparacion (8'): cada uno arma su historia de 10 lineas en notas minimas, una linea por forma como minimo. Esta es la base de "My Story".
4. Ejecucion (18'): cada estudiante CUENTA su historia completa a su pareja; la pareja usa la self-check card (5 preguntas). Cambio de parejas, segunda ronda mas fluida y sin fugas al presente.
5. Corrige en board solo errores en 2+ parejas (anonimos). Nombres a tu cuaderno.

4. Debrief (5'): observers reportan TIPOS ("all five present", "the earlier event slipped to simple") — nunca personas.
