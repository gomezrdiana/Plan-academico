## BLOQUE 2 — REEMPLAZA EL NOMBRE (40') — con el texto HEIIU "Mentors"

**Objective: by the end, everyone replaces names with the 7 object pronouns out loud, and finds them in a real text.**

2. Board (uso solo oral — Grammar formaliza hoy):
```
SUBJECT -> OBJECT (after the verb / after a preposition)
   I->ME  you->YOU  he->HIM  she->HER  it->IT  we->US  they->THEM
   "I know Camilo."  -> "I know HIM."
   "Can you help my mother?" -> "Yes, I can help HER."
   "I like pizza." -> "I like IT."   "I like hamburgers." -> "I like THEM."
Keep the ORDER of the sentence — only the noun becomes a pronoun.
```
