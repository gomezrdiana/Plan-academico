## BLOQUE 2 — HAD + PARTICIPLE (40')

**Objective: by the end, everyone nests an earlier event out loud with HAD + participle.**

1. Anuncia la meta en 1 linea (arriba). Board (uso oral — Grammar lo formaliza hoy mismo):
```
THE PAST BEFORE THE PAST — HAD + past participle
HAD never changes (I/you/he/we/they -> HAD + participle)
Regular participle = -ed (same as past simple): worked, played, finished
Irregular participle: gone, taken, seen, done, been, made
3 signal frames:
  "By the time + [past simple], + [HAD + participle]."
    -> "By the time the client called, I HAD finished the report."
  "[past simple] because + [HAD + participle]."
    -> "I was ready because I HAD prepared all week."
  "Before + [past simple], + [HAD + participle]."
    -> "Before the meeting started, I HAD sent the numbers."
Contrast: "When I arrived, it started" (same time) vs
          "When I arrived, it HAD started" (it happened first).
```
2. Modela 40 seg: tu historia con 1 evento principal (simple) + 2 flashbacks (had + participle). Señala: *"HAD + participle = the earlier event; past simple = the main event."*
3. Practica en parejas (10'): cada uno arma 4 lineas (1 main event + 2 had-flashbacks + 1 signal frame) sobre un logro/recuperacion real. La pareja verifica: ¿HAD + participio? ¿el flashback es lo que paso ANTES?
4. Ejecucion en circulo (16', de pie): cada estudiante DICE su main event + un flashback ("...because I had already..."); el siguiente pregunta ("What had you done before that?"). Rotacion, velocidad creciente.
5. Corrige en board solo errores en 2+ parejas (anonimos). Nombres a tu cuaderno.
