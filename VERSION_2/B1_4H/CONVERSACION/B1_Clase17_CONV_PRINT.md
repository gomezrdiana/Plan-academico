## BLOQUE 2 — USED TO / WOULD (40')

**Objective: by the end, everyone contrasts an old habit (used to) with today, out loud, correctly.**

1. Anuncia la meta en 1 linea (arriba).
2. Board (solo uso oral — Grammar lo formaliza hoy mismo):
```
USED TO / WOULD — the past you left behind
USED TO + BARE verb  = old habit OR state    "I used to work alone."
NEGATIVE:  DIDN'T USE to + BARE (no -d)       "I didn't use to speak up."
QUESTION:  DID + subj + USE to + BARE          "Did you use to smoke?"
WOULD + BARE = repeated ACTION only (never a state)
   "Every morning I would check the reports."  (action = OK)
   NOT "I would be shy."  -> use "I used to be shy."  (state)
Contrast:  "I used to work alone -> now I ask for help."
```
3. Modela 40 seg: 3 lineas "I used to..." verdaderas + 1 con would + el cambio a hoy. Señala: *"After USED TO the verb is BARE — same bare rule you met after DID in Cl 13."*
4. Practica en parejas (10'): cada uno arma 4 lineas (2 used to habit/state + 1 would + 1 contrast "used to X, now Y") sobre su trabajo/estudio real. La pareja verifica: ¿verbo bare? ¿would solo para accion?
5. Ejecucion en circulo (16', de pie): cada estudiante DICE una linea "I used to..." + su cambio; el siguiente lo pregunta ("Did you use to...?") y responde. Rotacion, velocidad creciente.
6. Corrige en board solo errores que aparezcan en 2+ parejas (anonimos). Nombres a tu cuaderno.
