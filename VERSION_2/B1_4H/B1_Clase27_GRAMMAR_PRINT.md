## BLOQUE 2 — DOS VERBOS, DOS FORMAS (40') — DE PIE ~90%

**Objective: everyone builds double-verb sentences on paper, choosing "to" vs "-ing" by the first verb.**

1. Board — cita del libro (M22, Guia 1 p.208-209):
```
GERUNDS WITH DOUBLE VERBS (A2 Book M22, p.207-209)
Regla 1: con DOS verbos, el segundo va en INFINITIVO o
         GERUNDIO. "I want to play." / "I like playing."
Regla 2: la forma la decide el PRIMER verbo.
Regla 3 (la clave):
  a. INFINITIVO (to + verb) = el 2do verbo NO ha empezado.
       "I want to play."  (aun no juego)
  b. GERUNDIO (verb + ing) = el 2do verbo SI ha empezado.
       "I like playing."  (ya juego)
BOOK TABLE (p.209):
  TO (not started):  decide to / forget to / need to /
                     want to / have to
  -ING (started):    start / finish / stop / like / love
BOOK EXAMPLES (p.209):
  "I want to wake up early."   "She doesn't have to eat."
  "He forgets to say hello."   "We decide to travel tomorrow."
  "I start studying."          "We finish talking."
  "I love eating ice cream."   "The girls stop fighting."
VOCAB (p.207-208): decide · forget (forgot) · start · stop
```
2. **Meta de tarea comunicativa (1 linea):** *"By the end of this block, this board holds 12 real work goals — half with 'to + verb', half with 'verb + ing', each in the right shape."*
3. **Preparacion en parejas (7', DE PIE junto al board):** cada pareja arma 1 meta con "to + verb" (want/need/decide) y 1 con "verb + ing" (love/start/stop/finish), reales de su trabajo.
4. **Ejecucion (12'):** cada pareja ESCRIBE sus 2 frases en el board (una zona por pareja), las dice, y el grupo verifica: ¿el primer verbo pide "to" o "-ing"? Meta: 12+ frases limpias.
5. **Errores tipicos (board, coro 2x):**
   - "I want learn" -> "I want TO learn" (want + infinitivo)
   - "I love to solve" -> "I love solvING" (love + gerundio)
   - "I stopped to check" (= me detuve para revisar) -> "I stopped checkING" (dejar de: love/stop + -ing)
6. **Producto verificable:** lectura coral del board; feedback primero sobre el logro (¿metas reales?), luego la forma.
