| **🎯 ABOUT THIS GUIDE**                                               |
|                                                                       |
| This guide provides a complete B2 program for 200 hours (4 hours/day, |
| Monday--Friday). It is designed for a young native-speaking teacher   |
| with limited formal experience. Two core innovations: (1) a 200-hour  |
| student project that gives meaning to the entire level, and (2) a     |
| Monday--Friday activity rotation system that prevents repetition,     |
| keeps energy high, and minimizes L1 (Spanish) use across 20 hours of  |
| class per week.                                                       |

1\. B2 Student Profile

A B2 student (CEFR) can understand main ideas of complex texts, interact
with native speakers with fluency, and produce clear, detailed texts on
diverse subjects. This program assumes students completed A2 and B1.

  -----------------------------------------------------------------------
  **Skill**         **B2 Target**
  ----------------- -----------------------------------------------------
  Listening         Understand extended speeches, news, films without
                    subtitles (familiar topics)

  Speaking          Debate, give reasoned opinions, narrate experiences
                    in detail

  Reading           Opinion articles, adapted literary texts,
                    professional documents

  Writing           Essays, formal emails, summaries of complex texts
  -----------------------------------------------------------------------

**CORE ELEMENT**

2\. The 200-Hour Student Project

Each student chooses a topic they're genuinely passionate about and
develops it across the full 200 hours. This is the most Montessori
element of the course: the student owns their learning.

2.1 Project Overview

  -----------------------------------------------------------------------
  **Element**             **Details**
  ----------------------- -----------------------------------------------
  Duration                200 hours (the entire B2 level)

  Topic                   Freely chosen by each student (must be approved
                          by teacher)

  Midpoint (Hour 100)     15-minute mini-presentation (10 min + 5 min
                          Q&A) + feedback

  Final (Hour 200)        30-minute presentation (20 min + 10 min Q&A)
                          with improvements from midpoint

  Written Deliverable     In-Class Process Portfolio (NOT a take-home
                          essay --- see Section 2.2)

  Language Req.           Must demonstrate B2 grammar, academic
                          vocabulary, coherent argumentation

2.2 In-Class Process Portfolio (Replacing the Take-Home Essay)

+-----------------------------------------------------------------------+
| **⚠️ WHY NOT A TAKE-HOME ESSAY?**                                     |
|                                                                       |
| A 1,500-2,000 word essay written at home is an open invitation to use |
| ChatGPT, Google Translate, or have someone else write it. At B2       |
| level, we need proof that the STUDENT wrote it, not a machine. The    |
| solution: all writing is done IN CLASS, in front of the teacher,      |
| building up piece by piece across the 200 hours. The final portfolio  |
| will contain more writing than a single essay, but every word was     |
| visibly produced by the student.                                      |
+-----------------------------------------------------------------------+

Instead of one final essay, each student builds an In-Class Process
Portfolio --- a physical binder or folder that grows throughout the
course. Every piece is written by hand or on a classroom computer during
class time, under teacher observation.

+-----------------------------------------------------------------------+
| **💡 WHY THIS WORKS BETTER THAN AN ESSAY**                            |
|                                                                       |
| 1\) Every piece is written in class --- the teacher SEES the student  |
| writing. No ChatGPT possible. 2) The portfolio shows GROWTH over      |
| time. You can literally see their English improving from Week 3 to    |
| Week 20. 3) It's more writing than a single essay (total:             |
| \~1,800-2,400 words across all pieces), but spread across months so   |
| it never feels overwhelming. 4) The reflective entries build          |
| metacognition: students think about HOW they're learning, not just    |
| WHAT. 5) Handwritten entries are especially powerful --- they're      |
| undeniably authentic.                                                 |
+-----------------------------------------------------------------------+

Teacher Rules for the Portfolio:

1.  **All writing happens in class.** No exceptions. If a student misses
    a writing session, they do it during the next project time slot.

2.  **Handwritten OR classroom computer only.** If students use
    computers, they write in a shared Google Doc where you can see the
    revision history, or they write on a classroom computer you can
    monitor.

3.  **Collect and store portfolios.** Portfolios stay in the classroom
    in a designated folder/box. Students don't take them home until the
    end of the course.

4.  **Date and sign each entry.** The teacher initials and dates each
    piece when it's completed. This creates a verified timeline.

**KEY INNOVATION**

3\. Monday--Friday Activity Rotation System

+-----------------------------------------------------------------------+
| **⚠️ THE PROBLEM WITH 20 HOURS/WEEK**                                 |
|                                                                       |
| 4 hours/day, 5 days/week = 20 hours of English. If every day looks    |
| the same, students get bored, the energy drops, and they default to   |
| speaking Spanish in pairs. The solution: each day of the week has a   |
| DIFFERENT personality. The 4-block structure stays the same, but the  |
| activities within each block change depending on the day.             |
+-----------------------------------------------------------------------+

3.1 Weekly Theme by Day

  -----------------------------------------------------------------------
  **Day**     **Theme**        **What Makes This Day Different**
  ----------- ---------------- ------------------------------------------
  Monday      NEWS DAY         All activities revolve around a single
                               news article. Reading, vocab, debate,
                               grammar from the article, writing a
                               response.

  Tuesday     SKILLS LAB       Focused practice on one specific skill
                               (rotate weekly: listening, writing,
                               reading, speaking). Deeper, slower work.

  Wednesday   PROJECT DAY      Extended project work time. Students work
                               on their 200-hour project. Teacher acts as
                               consultant.

  Thursday    GAME & CHALLENGE Competitive, high-energy activities:
              DAY              quizzes, team debates, role-play
                               competitions, grammar races.

  Friday      REAL WORLD DAY   Authentic tasks: write a real email,
                               simulate a job interview, watch and
                               discuss a TED Talk, analyze a real
                               advertisement.
  -----------------------------------------------------------------------

+-----------------------------------------------------------------------+
| **💡 WHY THIS ROTATION WORKS**                                        |
|                                                                       |
| 1\) Students wake up knowing what KIND of day it is. \"It's Thursday  |
| --- game day!\" creates anticipation. 2) Different days activate      |
| different energy levels: high-energy Thursday prevents mid-week       |
| slump. 3) Wednesday project day gives dedicated time without cramming |
| it into every lesson. 4) It prevents the #1 problem: predictability.  |
| When students know exactly what's coming, they tune out.              |
+-----------------------------------------------------------------------+

3.3 Strategies to Minimize Spanish Use

+-----------------------------------------------------------------------+
| **🚫 THE SPANISH PROBLEM**                                            |
|                                                                       |
| When two B2 Colombian students are put in a pair and told to          |
| \"discuss,\" the natural default is Spanish. You cannot prevent this  |
| with rules alone. You prevent it by making English the EASIER option  |
| through task design.                                                  |
+-----------------------------------------------------------------------+

Structural Solutions (Built into every day):

1.  **Written output before speaking:** Before any discussion, students
    must write something first (notes, answers, opinion sentences).
    Writing forces English. Then they SHARE what they wrote, which is
    already in English.

2.  **Task cards and sentence starters:** Give physical cards with
    phrases: \"I agree because\...\", \"On the other hand\...\", \"My
    opinion is that\...\". Students hold the cards and must USE them.
    This scaffolds the English and makes it easier than switching to
    Spanish.

3.  **Public performance pressure:** If students know they'll present
    their pair's discussion to the class, they prepare in English
    because they'll be embarrassed to present in Spanish.

4.  **The English Jar:** Every time someone speaks Spanish without
    permission (asking for a word is OK), they put a coin in the jar. At
    the end of the month, the jar money buys snacks for the class. Keep
    it fun, not punitive.

5.  **Physical movement:** Change the room layout. Gallery walks,
    standing debates, station rotations. Movement breaks the \"comfort
    zone\" that leads to Spanish chatting.

6.  **Teacher proximity:** Walk constantly. Don't sit at your desk. When
    you're near a pair, they speak English. When you're far away, they
    don't. Be everywhere.

7.  **Legitimate L1 moments:** Allow 2-3 \"Spanish minutes\" per class
    for genuine confusion. This paradoxically REDUCES overall Spanish
    use because students save it for when they really need it instead of
    defaulting to it.

5\. Your Guide as a Native Teacher

+-----------------------------------------------------------------------+
| **⚠️ COMMON NATIVE TEACHER MISTAKES**                                 |
|                                                                       |
| 1\) Speaking too fast. B2 ≠ native. Slow to 80%. 2) Not preparing     |
| because \"I can wing it.\" The rotation system requires preparation.  |
| 3) Correcting every error live. Kills fluency. Save for Error         |
| Correction time. 4) Can't explain grammar. Study each topic BEFORE    |
| class (use Murphy's English Grammar in Use).                          |
+-----------------------------------------------------------------------+

5.1 Pre-Class Routine (30 min)

-   Check what DAY it is (News/Skills/Project/Game/Real World) and
    review the plan.

-   Prepare day-specific materials (article, game cards, TED Talk link,
    role-play scenarios).

-   Review grammar topic if applicable.

-   Check which students need project check-ins (Wednesdays).

-   Prepare sentence starters/task cards for pair work.

5.2 Golden Rules

1.  **80/20 Rule:** Students 80%, you 20%. If you've talked 10+ min
    straight, stop.

2.  **Don't correct on the spot:** Note errors. Address later.
    Interrupting kills fluency.

3.  **Clear, short instructions:** What + how long + expected result.
    \"15 minutes. Pairs. 5 sentences. Third conditional. Go.\"

4.  **Be honest:** \"Let me check\" beats an incorrect explanation.

5.  **Use ICQs:** \"What are you doing first?\" Ensures understanding.

6.  **Move constantly:** Never sit at your desk during activities. Your
    presence = English use.
