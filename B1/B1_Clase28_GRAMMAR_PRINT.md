## 7. WALKING TRANSFORMATION — 16 min — STAND UP

Two zones in the room (tape labels on opposite walls):
- **WRONG zone (left):** color before size / origin before age / quantity in the middle / 5+ random adjectives
- **RIGHT zone (right):** correct CANT → OPIN → TAM → EDAD → FORMA → COLOR → ORIGEN → MATERIAL → NOUN

Hand each student **ONE Set D slip card** with a wrong-order sentence.

### Round 1 — Walk it — 8 min

Each student:
1. Stands in WRONG zone, reads slip aloud with the wrong order ("I have a black big car...").
2. Walks to RIGHT zone shouting the corrected form ("I have a big black car..."), naming the categories aloud ("size + color, correct").
3. Cohort confirms with thumbs.

Suggested slips:

```
"I have a black big car."
   → I have a big black car.                  (SIZE + COLOR)
"I want a white big beautiful house."
   → I want a beautiful big white house.       (OPIN + SIZE + COLOR)
"She has hair blond straight long."
   → She has long straight blond hair.         (SIZE + SHAPE + COLOR)
"It is a German red old wallet."
   → It is an old red German wallet.           (AGE + COLOR + ORIGIN)
"We need a wooden small old desk."
   → We need a small old wooden desk.          (SIZE + AGE + MATERIAL)
"The plane is a Brazilian large aircraft."
   → The plane is a large Brazilian aircraft.  (SIZE + ORIGIN)
"I have eyes small black."
   → I have small black eyes.                  (SIZE + COLOR; noun at END)
"There are metal 12 school desks."
   → There are 12 metal school desks.          (QTY + MATERIAL + USE/NOUN)
```

### Round 2 — Reverse — 4 min

Swap slips. Each student starts in RIGHT, walks to WRONG saying out loud WHY it's wrong: *"'black big' is wrong — SIZE comes BEFORE COLOR: 'big black', not 'black big'. The book sequence is quantity, opinion, size, age, shape, color, origin, material, noun — page 313."*

### Close — 2 min

2 volunteers walk their transformation in front of the group. Cohort counts correct sequences + names categories. Note errors with NAMES in your personal notebook.

## 9. HUMAN LINE-UP + 3 STATIONS — 12 min — STAND UP

### Human line-up — 4 min

**Use Set A category cards + Set B adjective cards + Set C noun cards.** Each student gets one adjective + its category card OR a noun + the NOUN card. You call out a target sentence; students walk to the front and stand in the correct word-order (respecting CANT → OPIN → TAM → EDAD → FORMA → COLOR → ORIGEN → MATERIAL → NOUN). Category cards make the order visible. **If two students have the same category, only one enters (TEMPERANCE — no doubling)**; if nobody has COLOR, no COLOR enters. 2–3 rounds, alternating combinations:

- SIZE + COLOR + NOUN: "big" + "black" + "car" → "a big black car"
- OPIN + SIZE + COLOR + NOUN: "beautiful" + "big" + "white" + "house"
- QTY + OPIN + AGE + NOUN: "2" + "beautiful" + "young" + "children"
- SIZE + AGE + MATERIAL + NOUN: "small" + "old" + "wooden" + "desk"

After each round: cohort reads the line aloud. Reshuffle cards. Next round.

### 3 stations — 8 min — bell every ~2.5 min

```
STATION A — SIZE + COLOR (TWO adjectives)
Build 4 sentences about your workplace with SIZE +
COLOR in the correct order (only 2 adjectives):
"I have a big black printer at home."
"We need a small grey company laptop."
"She wears a long red coat to meetings."
"The office has 3 small white desks."

STATION B — OPINION + SIZE + COLOR (THREE adjectives)
Build 4 sentences with OPIN + SIZE + COLOR:
"He drives a beautiful big black car to work."
"We bought a nice small white printer for the team."
"She has a beautiful long red coat for winter."
"They built a beautiful big white meeting room."

STATION C — AGE + COLOR + ORIGIN (THREE adjectives)
Build 3 sentences with AGE + COLOR + ORIGIN:
"It is an old red German wallet found in reception."
"He drives a new black Japanese motorbike to the office."
"We bought a new white Italian coffee machine for the kitchen."
```

You circulate, correct softly (especially wrong order or more than 3 adjectives = TEMPERANCE breach), note errors with NAMES.

You say a phrase with adjectives out of order (3 max); the next student responds with the phrase in correct order + names the categories:

```
"black big car"             → "a big black car"               (SIZE + COLOR)
"German old wallet"         → "an old German wallet"          (AGE + ORIGIN)
"blond long straight hair"  → "long straight blond hair"      (SIZE + SHAPE + COLOR)
"plastic small black phone" → "a small black plastic phone"   (SIZE + COLOR + MATERIAL)
"German tall serious man"   → "a tall serious German man"     (SIZE + OPIN + ORIGIN)
"red 2 nice cars"           → "2 nice red cars"               (QTY + OPIN + COLOR)
"old big house"             → "a big old house"               (SIZE + AGE)
"interesting little box"    → "an interesting little box"     (already correct: OPIN + SIZE)
```
