## BLOCK 2 — CONTENT TABLE + WOULD + vs PAST SIMPLE (~50 min) — STAND ~80%

```
1. ESCRIBE la tabla completa en board (toma 5 min,
   vale la pena — tu de pie todo el bloque):

   USED TO / WOULD — THREE MOVES + WOULD + CONTRAST

   AFFIRMATIVE — USED TO + BARE verb (habit OR state)
      I used to work nights.       (habit)
      She used to live in Bogotá.  (state)
      We used to have a small office. (state)
      They used to smoke.          (habit)

   NEGATIVE — DIDN'T USE to + BARE (no -d)
      I DIDN'T USE to work nights.   (not "didn't used to")
      She DIDN'T USE to live there.
      We DIDN'T USE to have an office.

   QUESTION — DID + subj + USE to + BARE (no -d)
      DID you USE to work nights?
      DID she USE to live there?
      DID they USE to smoke?

   WOULD — repeated ACTIONS only (not states)
      Every morning I WOULD arrive at 7.
      He WOULD check the mail before coffee.
      We WOULD close at 8 every night.
      *** NOT "I would live there" — state -> used to. ***

   USED TO vs PAST SIMPLE
      USED TO = repeated habit / extended state
         "I used to work nights."
      PAST SIMPLE = one finished event
         "Last night I worked late."

2. SEÑALA bloque AFFIRMATIVE (Stand). DI: "USED TO +
   BARE verb. Works for a HABIT you repeated and for a
   STATE that was true and is not anymore. 'I used to
   work nights.' 'She used to live in Bogotá.' Habit or
   state — both used to + bare."

3. [FRASE] insertion 2 (Stand). SEÑALA la frase. DI:
   "USED TO + bare holds the habit, DIDN'T USE to denies
   it, DID you USE to asks it, and WOULD repeats the
   action but never the state. Bare is courage made
   clean."

4. SEÑALA bloque NEGATIVE (Stand). DI: "Negative drops
   the -d. 'Didn't USE to.' Not 'didn't used to'. After
   DIDN'T the verb is bare — you proved this in Cl 13.
   'I didn't use to work nights.'"

5. SEÑALA bloque QUESTION (Stand). DI: "Question drops
   the -d too. 'Did you USE to work nights?' Bare USE
   after DID. Same machine as Cl 13 — after DID the
   verb returns home."

6. SEÑALA bloque WOULD (Stand). DI: "WOULD + bare for
   repeated ACTIONS. 'Every morning I would arrive at
   7.' But a STATE — live, have, be, own, like — does
   NOT take would. 'I would live there' is wrong. Say
   'I used to live there.' Actions take would; states
   stay with used to."

7. SEÑALA CONTRAST (Stand). DI: "If it happened ONCE
   and finished — past simple. 'Last night I worked
   late.' If it was a habit you repeated or a state
   that lasted — used to. 'I used to work nights.'"

8. [ERROR TIPICO #1] "I used to worked there"
   -> TACHA "worked", ESCRIBE "I used to WORK there."
   DI: "After USED TO = bare. Drop the -ed." COHORTE
   repite 2 veces en coro, de pie.

9. [ERROR TIPICO #2] "She didn't used to live here"
   -> TACHA "used", ESCRIBE "She DIDN'T USE to live here."
   DI: "Negative drops the -d. Bare USE after DIDN'T."
   COHORTE repite 2 veces.

10. [ERROR TIPICO #3] "Did you used to smoke?"
    -> TACHA "used", ESCRIBE "DID you USE to smoke?"
    DI: "Question drops the -d. Bare USE after DID.
    Cl 13 rule." COHORTE repite 2 veces.

11. [VIRTUD] activation (Stand): cuando alguien titubee
    entre la forma marcada y la bare, DI suave:
    "Fortitude. After USED TO the verb is bare. After
    DIDN'T it's USE. WOULD is for actions, not states."

12. [ERROR TIPICO #4] "I would live in Cali"
    -> TACHA "would live", ESCRIBE "I USED TO live in Cali."
    DI: "Live is a STATE. States don't take would.
    Used to." COHORTE repite 2 veces.

13. [ERROR TIPICO #5] "Yesterday I used to call him"
    -> TACHA "used to call", ESCRIBE "Yesterday I CALLED him."
    DI: "Yesterday = one finished event. Past simple,
    not used to." COHORTE repite 2 veces.

14. PIDELES en parejas DE PIE: "Transform 10 sentences.
    I dictate the source — some go to USED TO negative,
    some to USED TO question, some you fix the WOULD/
    state error, some you choose used to vs past simple."
    6 min. DICTA estos 10:
       a. "I used to worked nights"      -> FIX: "I used to WORK nights"
       b. "She used to live in Bogotá"   -> QUESTION: "DID she USE to live in Bogotá?"
       c. "They didn't used to smoke"    -> FIX: "They DIDN'T USE to smoke"
       d. "We used to have an office"    -> NEGATIVE: "We DIDN'T USE to have an office"
       e. "I would live in Cali"         -> FIX: "I USED TO live in Cali"
       f. "He used to print everything"  -> QUESTION: "DID he USE to print everything?"
       g. "Yesterday I used to call him" -> FIX: "Yesterday I CALLED him"
       h. "Every day I would arrive at 7"-> KEEP (correct — repeated action)
       i. "Did you used to work weekends?"-> FIX: "DID you USE to work weekends?"
       j. "We used to closed at 8"       -> FIX: "We used to CLOSE at 8"
    CIRCULA de pie. CORRIGE en board cuando un error
    aparezca 2 veces.

15. PODIUM — USED TO / WOULD SPRINT (7 min). DOS
    volunteers al frente, DE PIE. Tu das un kit
    (SUBJECT + VERB + HABIT/STATE/ACTION tag) y
    construyen la frase completa, manteniendo bare.
    Cohorte verifica en coro: "Confirmed — bare!"

    8 kits para llamar:
       I / work nights / HABIT      -> "I used to WORK nights"
       she / live there / STATE     -> "She used to LIVE there"
       you / smoke / QUESTION       -> "DID you USE to smoke?"
       we / have an office / NEGATIVE -> "We DIDN'T USE to have an office"
       he / arrive at 7 / ACTION-WOULD -> "He WOULD arrive at 7"
       they / own a shop / STATE    -> "They used to OWN a shop"
       you / work weekends / QUESTION -> "DID you USE to work weekends?"
       we / close at 8 / ACTION-WOULD -> "We WOULD close at 8"

    Plus 2 trap challenges:
       I / live there / WOULD?      -> "NO — used to LIVE (state)!"
       yesterday / call him / USED TO? -> "NO — CALLED (one event)!"

16. Aplauso al final. CIERRA cuando 80% construya used
    to + bare, didn't USE to, Did you USE to, y separe
    would-action vs used-to-state sin titubear. DI:
    "Stand. Push chairs to the walls. We go to the live
    sprint."
```
