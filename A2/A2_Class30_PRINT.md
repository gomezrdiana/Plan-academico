## ⚠️ NOTA SOBRE EL CONTENIDO — M30 TOO MUCH OR JUST ENOUGH?

**M30 oficial (libro pág 266-272):** "Too Much or Just Enough?" — Required Levels. Objetivo: decir si una cualidad llega a (o excede) el nivel esperado, sin dudar — con **enough** (nivel suficiente), **so** (nivel alto pero bueno) y **too** (nivel excesivo, no deseable).

Vocabulario base (descriptores reusados pág 268-269, ya conocidos): nice / new / old / good / bad / smart / rich / pretty / ugly / serious / early / late / complicated / important / beautiful / interesting / long / red / young / strong / dark / easy / hard / tall

```
CONCEPTO — El nivel de una cualidad: ¿suficiente, alto-bueno o excesivo?

LAS 3 PIEZAS DEL MÓDULO (libro pág 268):

  enough  → "enough" va DESPUÉS del descriptor (es inusual).
            "___ enough"      = nivel SUFICIENTE  → "I am tall enough."
            "not ___ enough"  = NO suficiente     → "I am not tall enough."

  so      → la cualidad es de nivel ALTO pero BUENO (positivo).
            "so + descriptor" → "You are so tall!"

  too     → la cualidad está EXCESIVA, hasta un punto NO deseable.
            "too + descriptor" → "You are too tall."

CONTRASTE clave del libro (pág 269):
  "She is so nice!"  (bueno)   vs.  "She is too nice."  (no puede pelear)
  "It is so new!"    (bueno)   vs.  "Your car is too new."  (yo quiero clásicos)
  "He is so smart!"  (bueno)   vs.  "He is too smart."  (no enseña a niños)
  "She is so serious." (bueno) vs.  "She is too serious." (no es feliz)

POSICIÓN: "enough" va DESPUÉS del adjetivo. "so" y "too" van ANTES.
   ✅ "tall enough"   ✅ "so tall"   ✅ "too tall"
   ❌ "enough tall"   ❌ "tall so"   ❌ "tall too"
```

**Conexión M28 → M29 → M30:**
- Cl 28 M28: *"X is THE tallest"* (el extremo, cierre del trío comparativo)
- Cl 29 M29: *"How tall is X? — He's pretty tall."* (el GRADO subjetivo de una cualidad)
- HOY M30: *"Is X tall enough? — He's too tall."* (si la cualidad LLEGA o EXCEDE el nivel requerido)

> M30 NO es comparación ni grado libre: mide si la cualidad alcanza el nivel JUSTO. Si dicen *"He is enough tall"* → corta: "enough" va DESPUÉS del adjetivo. Si mezclan "too" con "than" → corta: M30 NO lleva "than".

**OBJETIVO:** *"Open Module 30, page 266. Required levels. We do NOT grade freely anymore — we say if a quality is ENOUGH (just right), SO (high but good) or TOO (excessive, not wanted). 'enough' goes AFTER the adjective."*

### Vocabulario (5 min)

```
M30 VOCABULARY — descriptores (páginas 268-269, ya conocidos)
nice / new / old / good / bad / smart / rich / pretty / ugly /
serious / early / late / complicated / important / beautiful /
interesting / long / red / young / strong / dark / easy / hard / tall
```

Lee. Repiten 2x. Énfasis: "enough" /ɪˈnʌf/ (suena "i-NÓF", la "gh" no suena), "beautiful" /ˈbjuːtɪfʊl/.

### Concepto (10 min)

**Tablero:**

```
ENOUGH (after the descriptor):
   ___ enough        = just the right level   "I am tall enough."
   not ___ enough    = NOT enough             "I am not tall enough."

SO (before, high but GOOD):
   so + ___          "You are so tall!"   "She is so nice!"

TOO (before, EXCESSIVE / not wanted):
   too + ___         "You are too tall."  "She is too nice." (can't fight)

NO "than". NO "the ___est". M30 is NOT comparison — it asks if a
quality REACHES or EXCEEDS the required level.
```

**Ejemplos modelados** (libro pág 269, inserción #1 Frase del Día):
- *"We love Gabriela. She is so nice!" vs. "Gabriela can't fight. She is too nice."*
- *"Einstein is so smart!" vs. "Einstein can't teach kids. He is too smart."*
- *"You are never late. You are so early." vs. "My class at 5:00 — it is too early."*
- *"Am I patient enough? I'm not too anxious — I'm calm enough to wait."* ← Frase del Día
