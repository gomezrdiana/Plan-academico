## ⚠️ NOTA SOBRE EL CONTENIDO — M28 THE SUPERLATIVE

**M28 oficial (libro pág 253-255):** "You're the Best!" — The Superlative. Objetivo: formar el superlativo sin dudar.

Vocabulario base (pág 253): rich / pretty / ugly / serious / early / tall / complicated / nice / intelligent

```
RULE 1 — Short adjectives (1 syllable): THE + adj + -EST
  old → the oldest / tall → the tallest / rich → the richest
  Spelling: big → the biGGest / hot → the hottest

RULE 2 — 2 syllables ending in -y: THE + adj (y → -iest)
  pretty → the prettiest / ugly → the ugliest / early → the earliest

RULE 3 — Long adjectives (2+ syllables): THE MOST + adj
  intelligent → the most intelligent / serious → the most serious
  complicated → the most complicated

IRREGULARS: good → THE BEST / bad → THE WORST

Structure: "X is THE [adj-est]."  /  "X is THE MOST [adj]."
"Diego is the tallest."  /  "Spanish is the most complicated."
"She is the best."  /  "That was the worst."
```

**Conexión M26 → M27 → M28 (cierre del trío):**
- Cl 24 M26: *"X is AS tall AS Y"* (igual)
- Cl 27 M27: *"X is TALLER than Y"* (desigual, 2 cosas)
- HOY M28: *"X is THE tallest"* (el extremo, 3+ cosas)

> El superlativo NO lleva "than". Es el punto máximo. Si dicen *"the tallest than"* → corta.
