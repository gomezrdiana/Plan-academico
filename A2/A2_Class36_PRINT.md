**Conexión M37 → M38:**
- Cl 35 M37: módulo previo del libro (auditado contra A2 Book.md)
- HOY M38: cuando hay MÁS DE UN adjetivo, el inglés los ordena en una secuencia tradicional (cantidad → opinión/tamaño → edad → forma → color → origen → material → sustantivo); el orden importa para sonar natural.

> M38 NO es escoger un adjetivo: es ORDENAR varios. Si dicen *"a black big car"* → corta: tamaño antes que color → **a big black car**. Si dicen *"a curly short black hair"* → corta: tamaño/largo antes que forma antes que color → **short curly black hair**. Si dicen *"a wooden small old box"* → corta: tamaño → edad → material → **a small old wooden box**. Si dicen *"a old big house"* → corta: tamaño antes que edad → **a big old house**. Si dicen *"a white beautiful big house"* → corta: opinión → tamaño → color → **a beautiful big white house**.
