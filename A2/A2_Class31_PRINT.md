## ⚠️ NOTA SOBRE EL CONTENIDO — M32 HOW MANY PEOPLE ARE THERE?

**M32 oficial (libro pág 273-282):** "How Many People Are There?" — Specifying Countable Quantities. Objetivo del libro: sin dudar, formar preguntas usando **"how many"** y responder correctamente, con vocabulario contable. (El libro pasa de Module 30 directo a Module 32 — **no existe Module 31**, ver NOTA INTERNA.)

Vocabulario base (libro pág 274, Vocabulario 1):

```
M32 VOCABULARY (libro pág 274)
  a person          → una persona
  people            → gente / personas  (plural IRREGULAR de "person")
  many = a lot of = lots of   → muchos (objetos contables)
  a few             → unos / algunos  (aprox. 3 – 5)
```

```
CONCEPTO — Preguntar y responder CUÁNTOS hay (elementos CONTABLES)

GUÍA 1 "How many?" (libro pág 275, reglas literales):

  1. "how many...?" significa "¿Cuántos...?"
  2. Se usa para elementos CONTABLES (ej. 1 perro, 2 perros).
  3. Va ANTES del elemento de interés.

  Pregunta con palabra interrogativa:
     How many + [elemento plural] + do + [sujeto] + have?
     → "How many dogs do you have?" → "¿Cuántos perros tienes tú?"

  4. También con "there are / are there":
     → "How many dogs are there?" → "¿Cuántos perros hay?"

  5. "many" / "a lot of" / "lots of" = "muchos" (contables)
     → "I have many friends. / I have a lot of friends."

  6. "a few" = "unos / algunos" (aprox. 3 – 5)
     → "I have a few friends." → "Tengo unos amigos."

RESPUESTAS (libro pág 277-282):
  - número exacto:  "There are 7 nurses."  "She has 2 markers."
  - irregular:      "There are 5 people."  ("people", no "persons")
  - muchos:         "There are many people. / a lot of / lots of"
  - unos (3-5):     "There are a few sandwiches."  "He has a few chairs."
```

**Conexión M30 → M32:**
- Cl 29 M29: *"How tall is X? — He's pretty tall."* (el GRADO de una cualidad)
- Cl 30 M30: *"Is X tall enough? — He's too tall."* (si la cualidad LLEGA o EXCEDE el nivel)
- HOY M32: *"How many X are there? — There are a few."* (CANTIDAD exacta de cosas CONTABLES)

> M32 NO es comparación ni grado ni nivel: cuenta CANTIDAD de cosas contables. Si dicen *"How many money...?"* → corta: "how many" SOLO con contables (el dinero es M33, otra clase). Si dicen *"persons"* → corta: el plural irregular es **people**. Si dicen *"How many friends I have?"* → corta: falta el ayudante (*"do you have?"* / *"are there?"*).

### Concepto (10 min)

**Tablero:**

```
HOW MANY...? = ¿Cuántos...? (only COUNTABLE: 1 dog, 2 dogs)

M32 is NOT comparison, NOT level — it is COUNT.
```
