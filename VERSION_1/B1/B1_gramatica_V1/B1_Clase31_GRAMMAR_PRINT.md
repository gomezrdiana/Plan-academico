## 7. WALKING TRANSFORMATION — 16 min — STAND UP

Two zones in the room (tape labels on opposite walls):
- **STATE** ("I am tired" — static description)
- **CHANGE** ("I am getting tired" / "I got tired" — the movement)

Hand each student a Set H slip with a STATE sentence.

### Round 1 — Walk it — 8 min

Each student:
1. Reads STATE aloud ("I am tired" / "She is angry" / "The project is complex" / "The team is quiet").
2. Walks to CHANGE zone, saying the version with the right CHANGE verb: "I am GETTING tired" (in progress) / "I GOT tired" (past) / "She GOT angry" / "The project IS GETTING complex" / "The team WENT quiet" / "He BECAME rich after the IPO".
3. Cohort confirms thumbs + names the type aloud ("GET — quick everyday" / "BECOME — slow permanent" / "GO — idiomatic" / "TURN — age/color").

Suggested STATE → CHANGE slips:

```
STATE: "I am tired by 6 PM."     → CHANGE: "I GET tired by 6 PM (every day)." OR "I GOT tired by 6 PM yesterday." [GET]
STATE: "She is angry."           → CHANGE: "She GOT angry on Friday." [GET past]
STATE: "The project is complex." → CHANGE: "The project IS GETTING complex this quarter." [GET continuous]
STATE: "She is the team's go-to person." → CHANGE: "She is BECOMING the team's go-to person." [BECOME]
STATE: "The team is quiet."      → CHANGE: "The team WENT quiet when the CEO joined." [GO idiomatic]
STATE: "Our supervisor is 50."   → CHANGE: "Our supervisor TURNED 50 last week." [TURN]
STATE: "I am sick."              → CHANGE: "I GOT sick last night." [book p.336]
STATE: "Camila is bored."        → CHANGE: "Camila GOT bored in the Spanish class." [book p.336]
STATE: "Miguel is tired."        → CHANGE: "Miguel GOT tired after the game." [book p.336]
STATE: "The dog is lost."        → CHANGE: "The dog GOT lost in Piedecuesta." [book p.336]
STATE: "iPhones are huge in the US." → CHANGE: "iPhones GOT huge in the US." [book p.336]
STATE: "She is interested in the class." → CHANGE: "She GOT interested in the class." [book rule 4 p.335]
```

> The "She is interesting" (GOT interesting — CAUSE-changed) vs "She is interested" (GOT interested — FEELER changed) trap connects M41+M42 (book p.335 rule 4).

### Round 2 — Reverse — 4 min

Now reverse: start from CHANGE, walk to STATE saying the result aloud ("'I got tired' — so now I AM tired" / "'She became a manager' — so now she IS a manager" / "'The team went quiet' — so now the team IS quiet"). Swap slips.

### Close — 2 min

2 volunteers walk their transformation in front of the group. Cohort counts correct + names which verb (GET / BECOME / GO / TURN) and why. Note errors with NAMES.
