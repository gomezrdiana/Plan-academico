### ☐ 00:13–00:31 | LIBRO: MÓDULO 17 — HAVE + ANY / SOME (18 min)

**⚠️ MÁXIMO 18 MINUTOS. NO TE PASES.**

**⚠️ CONCEPTO CLAVE — HAVE + ANY/SOME:**

```
AFIRMATIVO (positivo):     SOME
   "I have SOME pizza."   "She has SOME questions."

NEGATIVO:                   ANY
   "I don't have ANY pizza."   "He doesn't have ANY money."

INTERROGATIVO (pregunta):   ANY
   "Do you have ANY pizza?"   "Does she have ANY questions?"
```

**Conjugación del verbo HAVE en presente simple:**
- I / You / We / They → **HAVE**
- He / She / It → **HAS**

**Negativos:**
- I / You / We / They → **don't have**
- He / She / It → **doesn't have**

Di: "Open your book to Module 17 — 'Do You Have Any Pizza?' Read with your partner. 5 minutes."

**Primeros 5 min:** Leen en parejas. Camina **EN SILENCIO**. Si preguntan: "Read it again with your partner."

**Siguientes 13 min:** Siéntate con cada pareja. ELLOS te explican (tú NO explicas):

1. ¿Qué se usa en afirmativo: any o some? → **SOME**.
2. ¿Qué se usa en negativo e interrogativo? → **ANY**.
3. ¿Cómo se conjuga have con she/he? → **HAS**.
4. ¿Cómo se hace la negación con she? → **doesn't have**.
5. ¿Cómo se hace la pregunta con you? → **Do you have any...?**

Si no pueden, vuelven a leer.

**Modelado de transformaciones (escribe en el tablero):**
```
ACTIVE → NEGATIVE → QUESTION

"I have some friends."
"I don't have any friends."
"Do you have any friends?"

"She has some money."
"She doesn't have any money."
"Does she have any money?"
```

Di cada par. La clase repite 2 veces.
