## ⚠️ NOTA SOBRE EL CONTENIDO — M31 THE EXISTENCE OF THINGS

**M31 oficial (libro pág 343):** "Is There Any Bread? — The Existence of Things"

Estructura clave: **THERE IS / THERE ARE** + **SOME / ANY** para describir existencia (o inexistencia).

```
EXISTENCIA — THERE IS / THERE ARE

SINGULAR / INCONTABLE → THERE IS
PLURAL (varios)        → THERE ARE

AFIRMATIVO:
- "There is a car."          (singular, contable)
- "There is some salt."      (incontable → some)
- "There are some cars."     (plural → some)

NEGATIVO:
- "There isn't a car."
- "There isn't any salt."    (incontable → any)
- "There aren't any cars."   (plural → any)

INTERROGATIVO:
- "Is there a car?"          → "Yes, there is. / No, there isn't."
- "Is there any salt?"       → "Yes, there is. / No, there isn't."
- "Are there any cars?"      → "Yes, there are. / No, there aren't."

REGLA SOME / ANY:
- AFIRMATIVO → SOME
- NEGATIVO e INTERROGATIVO → ANY
```

Vocabulario base del módulo (libro pág 344): **salt, sugar, butter, a movie, a player, a cookie** (+ incontables ya conocidos: water, milk, bread, cheese).

### Concepto (10 min)

**Tablero:**

```
THE EXISTENCE OF THINGS — THERE IS / THERE ARE

SINGULAR / UNCOUNTABLE → THERE IS
PLURAL                 → THERE ARE

              AFFIRMATIVE        NEGATIVE              QUESTION
Singular      There is a car.    There isn't a car.    Is there a car?
Uncountable   There is some      There isn't any       Is there any
              salt.              salt.                  salt?
Plural        There are some     There aren't any      Are there any
              cars.              cars.                  cars?

SOME / ANY RULE:
- AFFIRMATIVE → SOME
- NEGATIVE & QUESTION → ANY

SHORT ANSWERS:
- "Yes, there is. / No, there isn't."
- "Yes, there are. / No, there aren't."
```

**Ejemplos modelados del libro pág 346** (usar Frase del Día — inserción #1):
- *"Is there a movie?"* → *"Yes, there is a movie. / No, there isn't a movie."*
- *"Are there any cookies?"* → *"Yes, there are some cookies. / No, there aren't any cookies."*
- *"Is there any sugar?"* → *"Yes, there is some sugar. / No, there isn't any sugar."*
- *"Are there any students in the class?"* → *"Yes, there are some students in the class."*

## 8. SIMULACIÓN — "THE KITCHEN INVENTORY" (00:58–01:23, 25 min) — DE PIE

**OBJETIVO:** *"Real-life scene. You check a kitchen / pantry before cooking and report what there is and what there isn't, so the team buys only what's missing — no excess. That's temperance."*

### Setup (3 min)

**Escenario:** Un cocinero/encargado revisa la despensa de un restaurante o casa antes de cocinar. Un asistente pregunta qué hay y qué falta; juntos hacen la lista de compras justa (ni de más, ni de menos).

**Roles:** encargado de cocina (revisa y reporta) + asistente (pregunta + anota la lista).

### Vocabulario forzado

- Mínimo 5 preguntas con *"Is there any... ? / Are there any... ?"*
- Mínimo 5 respuestas con *"There is some... / There isn't any... / There are some... / There aren't any..."*
- Estructura: *"Is there any milk?"* → *"No, there isn't any milk. We need some."* / *"Are there any eggs?"* → *"Yes, there are some eggs."*

### Modela (3 min)

> *Asistente: "Is there any bread?"*
> *Encargado: "Yes, there is some bread. We have enough."*
> *Asistente: "Are there any eggs?"*
> *Encargado: "No, there aren't any eggs. We need some — but only a few, not too many."*

### Round 1 (9 min) — Grupos de 4

Cada grupo tiene una "despensa" (papelitos con nombres de alimentos: salt, sugar, butter, milk, bread, cookies, eggs). El asistente pregunta, el encargado responde con there is/are + some/any y arman la lista justa.

### Round 2 (8 min) — Rotación de roles

Nuevo encargado. Nueva despensa.

### Cierre (2 min)

1 grupo representa frente al curso. Aplauso. Anotas errores con NOMBRE en tu libreta personal.
