## ⚠️ NOTA SOBRE EL CONTENIDO — M27 UNEQUAL COMPARATIVES

**M27 oficial (libro pág 247):** "I'm Smarter than Steve" — Unequal Comparatives.

Vocabulario base: rich / pretty / ugly / serious / early / late / complicated

```
RULE 1 — Short adjectives (1 syllable): add -ER + THAN
  tall → taller than / fast → faster than / rich → richer than
  Spelling: big → biGGer than / happy → happiER than

RULE 2 — Long adjectives (2+ syllables): MORE + ___ + THAN
  complicated → more complicated than
  serious → more serious than

IRREGULARS:
  good → BETTER than / bad → WORSE than / far → farther than

Structure: "X is [more] [adj][-er] than Y."
"I am taller than my sister."
"This project is more complicated than the last."
"Coffee is better than tea."
```

**Conexión M26 → M27:** Cl 24 vieron AS...AS (igual). Hoy ven -ER/MORE...THAN (desigual). Misma comparación, distinta forma.
