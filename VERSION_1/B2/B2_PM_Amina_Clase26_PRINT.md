**Board — RULES:**

```
CHAINING PRACTICE — NO PAPER — CL 26
UNIT: HOOK + SEAM + PILLAR 1

PHASE A — ALONE (10 min):
  Paper out. Look at END of HOOK and START of PILLAR 1.
  Write ONE planning line:
   "MY SEAM: closing marker = ___
             breath = 1 full second
             opening marker = ___
             bridge formal→casual = yes
             anchor word walking across = ___"
  Cover paper. Say HOOK+SEAM+PILLAR 1 from memory 3 times.
  Each slip: glance, recover, no penalty.
  Goal: 3 clean chained attempts.

PHASE B — PAIRS (15 min):
  Paper DOWN. A stands, says HOOK + SEAM + PILLAR 1.
  B listens ONLY for the seam and flags ONE:
    (1) seam atropellado / no real breath?
    (2) closing + opening markers present?
    (3) bridge formal→casual landed?
  Swap. Each does 3-4 times. Coaching on SEAM ONLY.

PHASE C — MINI-AUDIENCE (20 min):
  Each student STANDS, says HOOK + SEAM + PILLAR 1
  once from memory (~2-2.7 min).
  Group flags ONE seam observation (no advice).
  Teacher times silently, private notes.
  5 × ~4 min = 20 min.

SEAM RULE:
  ONE real second of breath between closing marker
  (HOOK) and opening marker (PILLAR 1), bridge
  formal→casual.
  NOT "and now I want to talk about..." NOT silence-
  too-long. ONE second, then PILLAR 1 enters fresh.

KAREN: HOOK 40-50 sec + seam + PILLAR 1 45-60 sec — same standard.

NO IMPROVISATION:
  Pick ONE seam (one closing marker + one opening
  marker + one anchor + one Set #19 word optional).
  Drill THAT one. Improvising every time = never own
  any of them.

NO PAPER:
  Once Phase B starts, paper face down on floor.
  Comes up ONLY if you freeze for 5 full seconds.
  One glance, one breath, back to memory.
```

## 11. CHAINING HOOK → PILLAR 1 NO PAPER — 45 min — STAND UP

Setup (~30 sec): each student with Cl 19 script in hand (HOOK + PILLAR 1 visible). Visible countdown. Blank sheet for seam planning line.

Instructions (30 sec): *"On a blank line, write: MY SEAM — closing marker (last beat of HOOK), one second of breath, opening marker (first beat of PILLAR 1), anchor word walking across, optional Set #19 word. Teacher during 10 min: walk silent. DO NOT interrupt. Note in personal notebook (NOT communicated) who takes longest to write the planning line (signal: seam not clear, will be improvised). Note who covers paper fast and drills (signal: HOOK and PILLAR 1 individuals are solid).

### PHASE B — PAIRS (15 min)

Setup (~1 min): 2 pairs + 1 trio. Paper DOWN on floor.

Instructions (1 min): *"Paper face down. Partner A stands, says HOOK + SEAM + PILLAR 1 from memory. Partner B listens ONLY for the seam — flag ONE of three: breath real or rushed; closing/opening markers present; bridge formal→casual landed. Swap. Each does 3-4 times. Honest coaching on SEAM ONLY — do not touch HOOK or PILLAR 1 content."*

Teacher during 13 min: rotate silent between pairs (2-3 min each). Intervene ONLY if a pair drifts (correcting HOOK/PILLAR 1 content → *"Content closed since Cl 19. Today is the seam. Focus B's ear on closing marker, breath, opening marker. Go again."*). Note 3 errors with NAME per student.

## 14. ERROR PAPER LIVE — 5 min — STAND UP

```
Typical errors from chaining practice today:

1. ❌ "[HOOK ends] and now I want to talk about my first reason which is..."
   (dead seam, filler language, no breath, no casual bridge)
   → ✅ "[HOOK closing marker: '...and that is the picture I want you to hold.']
        (1 sec breath) Let me start where it matters most — the first reason is..."

2. ❌ "[HOOK] um, like, basically, yeah, so, [PILLAR 1 abrupt-academic]"
   (multiple fillers as seam substitute + PILLAR 1 enters rigid)
   → ✅ "[HOOK closing marker], (breath), so — let me bridge that — [PILLAR 1 casual]"
        (explicit bridge + casual + Set #19 'bridge' used)

3. ❌ HOOK closes with rising intonation (sounds like a question) → unstable seam → PILLAR 1 enters defensive
   → ✅ HOOK last sentence falling firm (declaration) → stable breath → PILLAR 1 fresh (not high, not low: new)

4. ❌ Inventing a new transition each attempt ("y ahora", "entonces", "ok pues", "right then") → 5 attempts = 5 different seams, none owned
   → ✅ One seam chosen in Phase A planning line + drilled in Phase B + delivered in Phase C (same one, refined)

5. ❌ Silence too long >2 sec that feels like "I forgot" (not temperance, freeze)
   → ✅ ONE second of breath, counted mentally, eye contact held (no audience lost)
```

Read paper (anonymous). Cohort corrects 2x — naming which LAYER failed (closing marker / opening marker / bridge / repetition of anchor / intonation) and which Set #19 word would tighten it (bridge / pivot / segue / link / transition / connect).
